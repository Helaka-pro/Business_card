MiCard - Flutter Personal Business Card App

A simple single-screen personal business card mobile application built using Flutter.
This project demonstrates fundamental Flutter layout and UI concepts while creating a clean and modern digital business card.

🚀 Features
Display profile image using CircleAvatar
Name and role with custom fonts
Contact information using Card & ListTile
Social media icons (Instagram, LinkedIn, GitHub)
Clean and centered layout using Column and Container
Material Design styling
Fast UI updates with Hot Reload

🧠 What I Learned
Flutter’s widget-based architecture
Layout building with Column and alignment properties
Styling widgets using Container and BoxDecoration
Adding custom fonts and image assets
Using Material Icons
Structuring a simple Flutter UI project

🛠️ Tech Stack
Flutter
Dart
Material Design

▶️ How to Run the Project
Clone the repository
git clone https://github.com/YOUR_USERNAME/micard_flutter.git
Navigate to the project folder
cd micard_flutter
Get dependencies
flutter pub get
Run the app
flutter run
